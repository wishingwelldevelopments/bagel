The Minimal Rice Manager

Bagel is a minimal shell script that allows you to connect to a host and remotely save your rice configurations

Step 1 : Download Bagel

You Can Use The Following Command To Download and Install The Bagel Script Into Your /usr/local/bin Directory

sudo wget https://git.io/JJJSc -O /usr/local/bin/bagel && sudo chmod +x /usr/local/bin/bagel

Step 2 Add The Sauce

If You Want My Dotfiles Feel Free. Use the following command :

bagel build
